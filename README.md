⛅ Weather App

Современное приложение для просмотра погоды в любом городе мира.  
Фокус на чистом UI, плавных переходах и работе с внешними API.
🌍 Возможности

    🔎 Поиск погоды по названию города

    🌡️ Текущая температура, ощущается как

    💨 Ветер, влажность, давление

    🕒 Локальное время города

    🎨 Адаптивный интерфейс

    ⚡ Быстрый отклик и оптимизация запросов

🛠️ Технологии

    Frontend: React / JavaScript

    API: OpenWeather API

    UI: Tailwind CSS / Custom CSS

    Инструменты: Git, VS Code, Linux

🎯 Цели проекта

    Работа с внешними API

    Управление состоянием в React

    Создание адаптивного интерфейса

    Практика асинхронных запросов
